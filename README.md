## 4. Công nghệ sử dụng
- Ngôn ngữ: C#
- Framework: .NET Framework
- Giao diện: Windows Forms
- Cơ sở dữ liệu: SQL Server
- IDE: Visual Studio 2013
- Quản lý mã nguồn: GitHub

## 5. Mô tả chức năng
- Đăng nhập hệ thống.
- Thêm nhân viên.
- Sửa thông tin nhân viên.
- Xóa nhân viên.
- Tìm kiếm nhân viên.
- Quản lý phòng ban.
- Thống kê danh sách nhân viên.

## 6. Hướng dẫn cài đặt
1. Cài đặt Visual Studio 2013.
2. Cài đặt SQL Server.
3. Clone project từ GitHub.
4. Khôi phục cơ sở dữ liệu vào SQL Server.
5. Mở file Solution (.sln).

## 7. Hướng dẫn chạy chương trình
1. Mở project bằng Visual Studio 2013.
2. Kiểm tra chuỗi kết nối SQL Server.
3. Nhấn F5 để chạy chương trình.
4. Đăng nhập và sử dụng các chức năng.
